readml.explainers.ml.shap_kernel_explainer.ShapKernelExplainer¶

class readml.explainers.ml.shap_kernel_explainer.ShapKernelExplainer(model: Any, features_name: List[str], classif: bool)[source]¶

Allows to explain globally or locally any non tree based model using Kernel SHAP method. Kernel SHAP is a method that uses a special weighted linear regression to compute the importance of each feature. The computed importance values are Shapley values from game theory and also coefficients from a local linear regression.

model¶

Trained model to interpret

Type:scikit-learn model or Pyspark model
