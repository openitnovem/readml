fbd_interpreter.explainers.dl.explain_dl.ExplainDL¶

class fbd_interpreter.explainers.dl.explain_dl.ExplainDL(model: Any, out_path: str)[source]¶

Class that contains different interpretability techniques to explain the DL model predictions depending on its data type and saves interpretability plots in out_path.

model¶

TensorFlow models and Keras models using the TensorFlow backend are supported Model to compute predictions using provided data.

Type

TensorFlow model

out_path¶

Output path used to save interpretability plots.
