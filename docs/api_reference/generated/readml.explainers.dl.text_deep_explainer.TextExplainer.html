readml.explainers.dl.text_deep_explainer.TextExplainer¶

class readml.explainers.dl.text_deep_explainer.TextExplainer(model: Any, target: str, word2idx: Dict[str, int])[source]¶

Allows to explain a DL model predictions on text data using shap deep explainer and a word to index mapping.

model¶

TensorFlow models and Keras models using the TensorFlow backend are supported Model to compute predictions using provided data.

Type:TensorFlow model
target¶

Target column name. If test_data doesn't have a target column, set target to ""

Type:str
word2idx¶

Dict where keys are the vocabulary and values the index.
