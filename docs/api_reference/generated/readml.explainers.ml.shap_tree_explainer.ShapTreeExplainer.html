readml.explainers.ml.shap_tree_explainer.ShapTreeExplainer¶

class readml.explainers.ml.shap_tree_explainer.ShapTreeExplainer(model: Any, features_name: List[str], classif: bool)[source]¶

Allows to explain globally or locally a tree based model using Tree SHAP algorithms. Tree SHAP is a fast and exact method to estimate SHAP values for tree models and ensembles of trees.

model¶

A tree based model. Following models are supported: XGBoost, LightGBM, CatBoost, Pyspark & most tree-based models in scikit-learn...

Type:scikit-learn model or Pyspark model
features_name¶

List of features names used to train the model

Type:List[str]
classif¶

True, if it's a classification problem, else False

Type:bool
