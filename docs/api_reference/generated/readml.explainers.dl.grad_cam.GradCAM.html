readml.explainers.dl.grad_cam.GradCAM¶

class readml.explainers.dl.grad_cam.GradCAM(model: Any, class_idx: int, layer_name: Optional[str] = None)[source]¶

Allows to explain a DL model predictions on image data.

model¶

TensorFlow models and Keras models using the TensorFlow backend are supported.

Type:TensorFlow or Keras model
class_idx¶

The class index used to measure the class activation map

Type:int
layer_name¶

layer to be used when visualizing the class activation map. If None, we try to automatically find the target output layer
