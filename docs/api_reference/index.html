API reference¶

This page gives an overview of all public classes, functions and methods.

Classes¶

Many classes can be used to interpret any model.

For ML interpretability:

explain_ml.ExplainML(model, task_name, ...) Class that contains different interpretability techniques to explain the model training (global interpretation) and its predictions (local interpretation).
shap_tree_explainer.ShapTreeExplainer(model, ...) Allows to explain globally or locally a tree based model using Tree SHAP algorithms.
shap_kernel_explainer.ShapKernelExplainer(...) Allows to explain globally or locally any non tree based model using Kernel SHAP method.

You can also use the icecream module classes for custom visualizations.

IceCream(data, feature_names, bins, ...) Class that generates and contains predictions and aggregations used to draw PDPlots, ICE plots and ALE plots.
IceCream2D(data, feature_x, feature_y, ...) Class that generates and contains predictions and aggregations used to draw 2D interaction plots (partial dependencies or ALE heatmaps).

For DL interpretability:

explain_dl.ExplainDL(model, out_path) Class that contains different interpretability techniques to explain the DL model predictions depending on its data type and saves interpretability plots in out_path.
tabular_deep_explainer.TabularExplainer(...) Allows to explain a DL model predictions on tabular data using Deep SHAP.
text_deep_explainer.TextExplainer(model, ...) Allows to explain a DL model predictions on text data using shap deep explainer and a word to index mapping.
image_deep_explainer.VisualExplainer(model) Allows to explain a DL model predictions on image data.
grad_cam.GradCAM(model, class_idx, layer_name) Allows to explain a DL model predictions on image data.
